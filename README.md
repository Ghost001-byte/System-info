# Progetto 1: Info Hardware e OS

Questo programma mostra le informazioni sull'hardware e sul sistema operativo del PC.

## Requisiti
- Python 3.x
- Libreria `psutil`

## Installazione dipendenze

Apri il terminale e digita:
```
pip install psutil
```

## Esecuzione

Esegui il programma con:
```
python "Progetto 1.py"
```
